# Resume ATS Analyzer

A free, open-source ATS (Applicant Tracking System) resume analyzer built with Python and Streamlit. Upload your resume, paste a job description, and get an instant ATS compatibility score with actionable improvement suggestions.

Most ATS scoring tools are behind paywalls — this project replicates that functionality and makes it freely accessible.

## Features

**ATS Score (0–100)** — weighted scoring across five dimensions
**Keyword Match Analysis** — TF-IDF + Cosine Similarity against the job description
**Section Detection** — checks for Experience, Education, Skills, Projects, and more
**Action Verb & Quantification Check** — flags weak bullet points
**Formatting Audit** — detects ATS-unfriendly formatting patterns
**AI-Powered Suggestions** — deep analysis via Groq API (Llama-3.3-70b)
**Bullet Point Rewrites** — before/after examples generated by the model
**Downloadable Report** — export your analysis as a `.txt` file

## Tech Stack
| Layer | Technology |
| UI | Streamlit |
| File Parsing | pdfplumber, python-docx |
| NLP / Scoring | scikit-learn (TF-IDF, Cosine Similarity), NLTK |
| AI Analysis | Groq API — Llama-3.3-70b-versatile |
| Visualization | Plotly |
| Language | Python 3.10+ |

## Scoring Methodology
| Module | Weight | Method |
| Keyword Match | 35% | TF-IDF vectorization + Cosine Similarity |
| Section Detection | 20% | Regex pattern matching |
| Formatting | 20% | Rule-based heuristics |
| Action Verbs | 15% | Custom verb lexicon + regex |
| Contact Info | 10% | Regex (email, phone, LinkedIn, GitHub) |
